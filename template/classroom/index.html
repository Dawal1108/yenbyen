Yenbyen Fellowship Class Manager

A Fellow-Tutor Portal

About Us

Yenbyen Fellowship Class Manager is a fellow-tutor portal with various classroom features like uploading assignments, submissions, Class Notice and many more. Fellows and Tutors can login in the portal and can experience interactive classroom experience online. One of the most powerful features of Yenbyen Fellowship class manager is its Fellow managing abilities. Tutor can easily manage their Fellows and keep them updated about all of the activities in class.

Our Services

For Tutor

  1. Tutors can add fellow in their class through Add Fellow option.
  2. After adding Fellows in their class, Tutor can give or edit their class fellows marks.
  3. Tutors can also write notice which will be send to all fellows in their class.
  4. Tutors can upload assignments through upload assignment options.
  5. Tutor can check their fellow's assignment submission and give them marks accordingly.
  6. Fellows can message Tutors directly and their messages can be seen in Inbox.
  7. Tutor can also see all the marks given by them to a fellow by accessing their Marks profile and can edit their marks.
  8. User can add profile pic through edit profile option.

For Fellows

  1. Fellows can login in the portal and keep themselves updated of all the activities going on in the class.
  2. Fellows can see the notice given by their tutor in the Class Notice option.
  3. Fellows can download their assignments in the Assignments option and can also submit it their. ( Once assignment is submitted, it can't be changed later. )
  4. Fellows can contact any tutor through message option in All Tutors List.
  5. Fellows can check their marks in the Marks option.
  6. User can add profile pic through edit profile option.
